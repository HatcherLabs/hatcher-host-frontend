## Maker's Comment

Hey Product Hunt!

I'm the solo dev behind Hatcher. I built it because I kept wasting days deploying AI agents that took minutes to configure.

Hatcher gives every agent its own Docker container with managed infrastructure. You pick a framework, set up your agent's personality and tools, connect platforms, and click deploy. That's it.

The free tier is real — 1 agent, all integrations, free Groq LLM (10 msg/day). Bring your own API key and you get unlimited messages even on the free plan.

Four frameworks supported today: OpenClaw (tool-heavy), Hermes (conversational), ElizaOS (character-driven), and Milady (personality presets). 23+ templates to start from.

For developers: we have an SDK, CLI, GitHub Action, Zapier integration, and full REST API.

Payments are in SOL/crypto — we're building for the crypto-AI intersection.

Would love your feedback. What frameworks or integrations should we add next?

Right now we support 4 frameworks with config-based setup. Custom code support is on the roadmap.
