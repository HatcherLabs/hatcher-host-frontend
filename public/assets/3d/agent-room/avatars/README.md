# Agent Room Avatars

Local GLB avatars used by the Hatcher agent room.

- `ready-player.glb` - three.js examples `readyplayer.me`.
- `xbot.glb` - three.js examples `Xbot`.
- `soldier.glb` - three.js examples `Soldier`.
- `cesium-man.glb` - Khronos glTF Sample Models `CesiumMan`.
- `astronaut.glb` - model-viewer shared assets `Astronaut`.
- `animated-robot.glb` - Quaternius `Animated Robot`, mirrored from Get3DModels public domain GLB listing.
- `freepixel-robot.glb` - FreePixel 3D assets `Robot`.
- `freepixel-alien.glb` - FreePixel 3D assets `Alien`.
- `freepixel-astronaut.glb` - FreePixel 3D assets `Astronaut`.
- `freepixel-ninja.glb` - FreePixel 3D assets `Ninja`.
- `freepixel-rogue.glb` - FreePixel 3D assets `Rogue`.
- `get3d-service-robot.glb` - Get3DModels `Robot - Runaway Boy Stealth Escape`.
- `get3d-rusty-mecha.glb` - Get3DModels `Baby's First Mecha`.
- `get3d-abandoned-mecha.glb` - Get3DModels `Abandoned Mecha`.
- `get3d-drone.glb` - Get3DModels `Drone`.
- `get3d-cyber-drone.glb` - Get3DModels `Cyberpunk Drone Concept Design`.
- `get3d-buzz-droid.glb` - Get3DModels `Buzz Droid`.
- `get3d-cybernetic-warrior.glb` - Get3DModels `Procedural Scifi Armor Test 7`.
- `blocky-agent.glb` - Kenney `Blocky Characters` character-a.
- `blocky-street-operator.glb` - Kenney `Blocky Characters` character-b.
- `blocky-green-technician.glb` - Kenney `Blocky Characters` character-c.
- `blocky-cyborg.glb` - Kenney `Blocky Characters` character-d.
